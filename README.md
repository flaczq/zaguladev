# Zagula Productions Portfolio

To jest profesjonalna strona wizytówka stworzona dla **Zagula Productions**. 

## Funkcje
- **Fancy Background**: Dynamiczne, ruchome ikony (pady, rakiety, przekładnie) z efektem paralaksy reagującym na mysz.
- **Glassmorphism**: Nowoczesny design karty głównej z rozmyciem tła.
- **Premium Aesthetics**: Gradienty, nowoczesna typografia (Outfit) i subtelne animacje.
- **Responsive Design**: Strona wygląda świetnie na komputerach i urządzeniach mobilnych.

## Jak używać?
1. **Avatar**: Zapisz swój obrazek (ten z postaciami w garniturach) jako `avatar.png` w tym folderze. System automatycznie go wykryje i oprawi w świecącą ramkę.
2. **Linki**: W pliku `index.html` znajdź sekcję `<div class="links">` i podmień `#` na swoje prawdziwe linki do Steam, Portfolio czy Social Media.

## Publikacja na GitHub Pages
1. Stwórz nowe repozytorium na GitHub (np. o nazwie `zaguladev`).
2. Wrzuć tam te pliki (`index.html`, `style.css`, `script.js` oraz swój `avatar.png`).
3. Wejdź w **Settings** -> **Pages**.
4. W sekcji **Build and deployment** wybierz branch `main` i folder `/ (root)`.
5. Kliknij **Save**. Twoja strona będzie dostępna pod adresem `https://TWOJ-LOGIN.github.io/zaguladev/`.
